## ‚≠ê PART 1 ‚Äî Basic Structure & Headings
Every HTML document follows this structure:

- `<!DOCTYPE html>` ‚Äî HTML5 declaration  
- `<html>` ‚Äî root element  
- `<head>` ‚Äî metadata, title, CSS  
- `<body>` ‚Äî visible content  

**Headings (`<h1>` to `<h6>`)** define title hierarchy.
